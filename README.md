# AI Assisted Mental Health Journaling App MVP
# Run and test locally

This contains everything you need to run your app locally.


## Run Locally

**Prerequisites:**  Node.js
Clone the repository in your local machine using `git clone https://github.com/carlosazurmendi/ai-journal-mvp.git` and CD into the repository folder.

Rename placeholder.env to .env.local and add your environment variables. (Right now, only your gemini API key is needed)

1. Install dependencies:
   `npm install`
3. Run the app:
   `npm run dev`
